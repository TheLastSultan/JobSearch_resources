# W2D3

## Pair Programming

### Length of Spans (10 mins)
Find the combined content length of all of the spans on this (link) page.

### Total Shares (30 mins)
Given an array of objects, each with keys that are the social media (i.e. facebook, instagram), and values that are # of shares (i.e. 2, 3), write a function that will return an object counting the total number of shares for each social media

### Byte Intersection (20 mins)
Given two integers, find the intersection of their byte representations

### String Calculator (40 mins)
Given a valid equation parsed as a string, return the actual numeric value of the equation. For simplicity, just assume the equation only has addition and multiplication.
'5+2*5+3*2' => 21
